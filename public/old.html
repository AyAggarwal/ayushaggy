Musical embedded systems applications. Using ESP32 w/ Rust.
AutoTheramin
Elegantly responsive instrument utilizing Hall Effect sensors and a magnet to detect near real-time changes in proximity. Software is programmed in bare-metal, no-std Rust and works with any midi output device. Stay tuned for a demonstration video!
Camera Theramin
modulates the pitch of an ongoing sine wave sound, making the hand's movement audibly perceptible. This creates an environment where the user's hand motions directly influence the system's auditory outputs, reminiscent of sensor based embedded systems.
